# dotfiles
## installation
```sh
curl -sSL https://raw.githubusercontent.com/serna37/dotfiles/master/brew.sh | sh \
    && curl -sSL https://raw.githubusercontent.com/serna37/dotfiles/master/install.sh | sh \
    && exec $SHELL -l
```
