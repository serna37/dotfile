# dotfiles
## installation
```sh
curl -sSL https://raw.githubusercontent.com/serna37/dotfiles/master/install.sh | sh
exec $SHELL -l
```

> [!Note]
> コードの署名[参考](https://blog.symdon.info/posts/1610113408/)

> [!Note]
> Finderのキル
> ```
> defaults write com.apple.Finder QuitMenuItem -boolean true
> ```
